# abbreviations

Fast typing with abbreviations c:

## changelog v 1.0.2

Now script logging some errors in debug.log\
Now script configurable with abrSettings.ini

### Requirements

```
Python 3.10
pip install pyperclip
pip install keyboard
pip install pynput
pip install loguru
```
